Formato PNG
  • El nombre del formato PNG corresponde a Portable Network Graphics y se presenta como el formato del futuro. Se trata de un formato gráfico concebido especialmente para la aplicación en la Web. En principio PNG debe reunir las ventajas del formato GIF y del formato JPEG.
  • Este formato acepta 16,7 millones de colores al igual que el formato JPEG.
  • Su compresión es elevada y se efectúa sin pérdida de datos y de calidad.
  • Ofrece la posibilidad de transparencia.
  • Su función entrelazada permite una visualización progresiva.
  • Con este formato no se pueden generar imágenes animadas.
  • Se prevé que este formato PNG se utilizará cada vez más ya que los profesionales lo han adoptado en lugar del GIF y para fotos pequeñas.
